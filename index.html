Projects

In the past three years, I've pushed myself to improve my technical skills. From writing my first line of code in 2018 to the present, I have worked on a variety of projects, applying my skills and learning new ones.

Log Manager View

A command-line interface written in C++ to manage structured logs. Curate an excerpt list of logs and search by keyword, category, or timestamp.

  • Parsed structured log file, applying std::unordered_map to cache data in memory.
  • Optimized performance by minimizing data redundancy and leveraging counting sort for specific user commands.

Wikipedia Search Engine *

A search engine for scraped Wikipedia data, serving results ranked by relevancy to user queries.

  • Developed MapReduce pipeline to create inverted index with parallelized data processing via Hadoop.
  • Created API server that queries inverted index, computes weighted average of TF-IDF and PageRank scores, and serves relevant hits.
  • Built front-end to display search form and results, issuing requests to API server and populating webpage details from an SQLite database.

Euchre *

A simulator for Euchre, a popular card game played with 4 players. Run with any combination of human and bot players.

  • Leveraged polymorphism and factory design pattern to hide implementation details of each type of player.
  • Created command-line interface to display each player's hand and allow human players to choose their move.

A* Search Visualizer View

A visualizer for the A* (A-star) search algorithm, written with Python and Pygame. Select start and end points, as well as obstacles via a graphical interface.

  • Visually distinguishes visited, unvisited, and enqueued nodes via color.
  • Implemented easy-to-use graphical user interface with Pygame, allowing user to modify search parameters, such as diagonal movement and obstacles.
